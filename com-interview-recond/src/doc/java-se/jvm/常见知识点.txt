内存分布
    1、new 开辟一块堆内存(存放类的属性)
    2、对象地址存放于栈内存(指向于堆内存的数据)
    3、static存放于全局数据区，static变量为类所有(注：可以通过对象改变、也可以通过类改变)
    4、方法存放于全局代码区

静态代码块：
    静态代码块只会执行一次，在首次加载类的时候执行(在构造方法之前执行)，一般会用来初始化一些值，并且在所有对象中全局共享
    静态代码块 优先于 普通代码块 优先于 构造方法

子类实例化：(在有父类的情况下，初始化执行顺序--注意：static至只执行一次)可以执行初始化工作
    ClassParent static
    ClassSon static
    ClassParent
    ClassSon
